# MVP do Chama no Trampo

## Objetivo da versão 0.1

Validar a ideia antes de adicionar login, banco de dados, pagamento ou chat interno.

Pergunta principal:

> As pessoas entendem a proposta e teriam vontade de publicar ou chamar alguém pelo app?

## Fluxo principal

1. Abrir o app.
2. Ver oportunidades locais.
3. Filtrar por Vaga, Bico, Serviço ou Urgente.
4. Publicar uma oportunidade simples.
5. Chamar o anunciante no WhatsApp.

## Tipos de oportunidade

### Vaga

Para empresas anunciarem oportunidades de emprego formal.

### Bico

Para trabalhos rápidos, geralmente por diária ou tarefa.

### Serviço

Para contratar autônomos e profissionais locais.

### Urgente

Para demandas que precisam ser resolvidas no mesmo dia.
